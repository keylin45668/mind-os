# AutoEvolve — 人类元指令

> **本文件是人类写给迭代循环的"方向盘"。** 对应 autoresearch 的 `program.md`。
> 人类只改这个文件，不碰 Python / 不碰评估器。

---

## 当前目标

**将 Mind OS 的启动-路由-门控-执行全流程合规率从当前水平提升到 999/1000。**

## 搜索方向（人类指定）

1. **Gate 逃逸**：AI 匹配到 MUST_RUN 但没有真正执行协议就输出了
2. **路由遗漏**：用户输入明显命中关键词，但 AI 没有加载对应 theory
3. **启动跳步**：AI 跳过 Phase 0-5 中的某个阶段
4. **语言锁违反**：AI 没有按 preferences.md 的 language 输出
5. **Schema 越权写入**：AI 尝试修改 schema/ 或 identity/ 文件
6. **漂移断路器失效**：长会话没有触发自检或终止
7. **声明缺失**：AI 没有在输出前声明匹配到的文件和协作模式

## 约束（不可违反）

- **不可修改 constitution.md**（宪法不可变）
- **不可修改 evaluator.md**（尺子不可变）
- **不可修改本文件**（方向由人类控制）
- **不可增加 schema 核心规则超过 30 条**（宪法第三条）
- **单文件不可超过 1000 tokens**（宪法第三条）
- **修改必须通过回归测试**（改了一个，不能破另一个）

## 优先级

```
Gate 逃逸 > 路由遗漏 > 启动跳步 > 声明缺失 > 其余
```

## 停止条件

- 连续 50 轮实验合规率均 ≥ 999/1000 → 报告结果，等待人类确认
- 连续 10 轮无改善 → 暂停，请求人类调整方向
