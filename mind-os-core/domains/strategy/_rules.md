# Domain: 战略与竞争

> 关键词：战略、竞争、市场、定位、壁垒、增长
> 默认协作模式：对等协作 × 迭代

---

## 1. 协作模式默认配置

| 任务类型 | 模式 | 拓扑 | 说明 |
|---------|------|------|------|
| 战略规划 | 对等协作 | 迭代 | 人定愿景，AI提供分析框架 |
| 竞争分析 | AI主导 | 串联 | AI收集整理，人判断战略含义 |
| 市场进入 | 对等协作 | 并联 | 双方独立分析后合并 |
| 定价策略 | 对等协作 | 迭代 | AI测算，人决策 |
| 业务复盘 | 对等协作 | 迭代 | AI整理数据，人提炼洞察 |

---

## 2. 战略思考框架

```yaml
analysis_layers:
  1. 本质追问: "这个市场的第一性原理是什么？"
  2. 五事七计: "道天地将法，谁的基本面更强？"
  3. 不战而胜: "有没有不正面竞争就能赢的路径？"
  4. 反脆弱检验: "市场波动对我是利还是弊？"
  5. 十倍压力测试: "规模×10，战略还成立吗？"
```

## 3. 战略专属偏差检查

```yaml
competitor_fixation:
  trigger: 过度关注竞争对手而非用户需求
  protocol: "回到用户——用户真正的痛点是什么？"

strategy_tax:
  trigger: 战略讨论超过2小时无可执行产出
  protocol: "停止分析，列出下周可以做的3件事"

survivorship_bias:
  trigger: 只看成功案例做类比
  protocol: "找3个失败案例，分析失败原因"
```
