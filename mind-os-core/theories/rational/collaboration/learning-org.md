## 深度汇谈 vs 讨论

| 模式 | 目的 | AI角色 |
|------|------|--------|
| 深度汇谈 | 探索未知，暴露假设 | 用推论阶梯提示暴露假设 |
| 讨论 | 比较方案，做出决策 | 用偏差检查辅助评估 |

切换规则：探索→汇谈，决定→讨论。AI 检测混淆时提醒。

## 双环学习

```yaml
single_loop: 行动→结果→调整行动（改method）
double_loop: 行动→结果→质疑假设→调整心智模型（改theory）
trigger: 单环学习连续3次未改善 → 强制进入双环
```

## 团队学习障碍识别

```yaml
barriers:
  - 熟练的无能：自动化行为阻碍反思 → 减法实验打破
  - 习惯性防卫：回避暴露思维过程 → 推论阶梯强制透明
  - 团体迷思：过度追求一致 → 并联拓扑+独立判断
```
