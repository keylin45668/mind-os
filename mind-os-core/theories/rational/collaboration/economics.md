## 比较优势原则

即使 AI 在所有任务上都更高效（绝对优势），双方仍应按比较优势分工——各自做机会成本最低的事。

```yaml
protocol:
  1. 列出所有任务类型
  2. 每个任务计算双方机会成本
  3. 分配给机会成本更低的一方
  4. 每季度重新校准（AI能力在变化）
```

## 自组织条件

当以下三个条件同时满足时，系统倾向自组织（最小介入）：

```yaml
self_organize:
  条件1: 充分信息（各方能看到相关数据）
  条件2: 自由选择（可自主决定参与方式）
  条件3: 可退出（不满意可撤回）
  全满足 → 允许自组织，AI最小介入
  任一不满足 → 需要系统调度
```

## 认知资本管理

```yaml
cognitive_capital:
  stock:
    - 知识库容量
    - 技能水平
    - AI模型版本
    - 协作默契度
  health_check: 认知利润(flow_in - flow_out)连续3期为负 → 资本收缩流程
  investment: 优先投资"利润率"最高的认知领域
```
