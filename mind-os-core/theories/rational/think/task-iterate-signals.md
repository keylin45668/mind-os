## 摘要

- **四项质量信号**：论点存活率(≥60%)、新发现数(应边际递减)、假设验证率(≥80%)、本源偏离度(≥4)
- **正常收敛**：连续2轮新发现=0 + 存活率变化<5% + 假设验证≥80%
- **强制停止**：达 max_rounds(5) / 偏离度<3 / 存活率<30%
- **异常退出**：R1 红方推翻核心问题本身→停止并重新定义

# 任务迭代 — 质量信号与收敛规则

---

## 质量信号（每轮必须计算并输出）

```yaml
quality_signals:
  论点存活率:
    计算: 存活论点数 / 总论点数
    健康值: ≥ 60%（低于=论证基础薄弱）

  新发现数:
    计算: 本轮新增论点 + 新发现风险 + 新回答问题
    趋势: 正常应边际递减

  假设验证率:
    计算: 已验证假设 / 总假设
    健康值: ≥ 80%

  本源偏离度:
    计算: 当前论点链与第1轮核心问题的相关性（AI自评 1-5）
    健康值: ≥ 4（<3 = 漂移）
```

## 每轮强制输出格式

```
── 迭代 R{N} ──────────────────
🔵 蓝方: {新增/强化了什么}
🔴 红方: {攻击了什么，首要=本源检查}
⚖️ 裁决: {SURVIVED/FALLEN/MODIFIED 各几条}
📊 信号: 存活率 {X}% | 新发现 {N} | 假设验证 {X}% | 偏离度 {1-5}
📈 趋势: {↑改善 / →持平 / ↓恶化}
────────────────────────────────
```

## 收敛条件

```yaml
convergence:
  正常收敛:
    - 连续 2 轮新发现数 = 0
    - 且论点存活率变化 < 5%
    - 且假设验证率 ≥ 80%

  强制停止:
    - 达到 max_rounds（5 轮）
    - 或本源偏离度 < 3
    - 或论点存活率 < 30%

  异常退出:
    - R1 红方推翻核心问题本身 → 停止，提示重新定义
```
