## 摘要

- **战前评估**：五事(道天地将法)七计对比，>1.2x可进入，±10%需差异化，<0.8x不建议正面竞争
- **博弈识别**：囚徒困境(价格战)/猎鹿(跨部门协同)/协调(技术选型)/懦夫(强硬谈判)
- **不战而胜**：伐谋→伐交→伐兵→攻城，有可行方案即停止升级
- **形势判断**：形(实力)×势(趋势)四象限

# 竞争策略方法论

---

## 战前评估（五事七计）

```yaml
五事: 道(使命-趋势对齐) / 天(宏观时机) / 地(赛道壁垒) / 将(领导力) / 法(流程制度)
七计对比: 每维度0-10分，生成雷达图
决策门槛:
  我方 > 对方×1.2 → 可进入
  我方 ≈ 对方(±10%) → 需差异化
  我方 < 对方×0.8 → 不建议正面竞争
```

## 博弈识别

| 博弈类型 | 特征 | 典型场景 |
|---------|------|---------|
| 囚徒困境 | 都背叛是纳什均衡 | 价格战 |
| 猎鹿 | 合作帕累托最优但需信任 | 跨部门协同 |
| 协调 | 多均衡，偏好不同 | 技术栈选型 |
| 懦夫 | 两纯策略均衡 | 强硬谈判 |

```yaml
重复博弈: 以牙还牙 + 三步纠错
信息不对称: 信号传递 + 筛选机制
承诺可信度: 锁定成本 + 第三方背书 + 历史一致性
纳什-帕累托比较: 有帕累托改进 → 优先谈判而非对抗
```

## 不战而胜阶梯

```
① 伐谋（策略解决）→ 重新定义赛道/标准制定
② 伐交（外交解决）→ 并购/联盟
③ 伐兵（竞争对抗）→ 须通过五事七计+虚实分析+选择权检查
④ 攻城（全面战争）→ 最后手段，须手动确认
逐级检查，有可行方案即停止升级。
```

## 虚实分析

识别对手强项(实→回避)和弱项(虚→集中3倍资源进攻)。

## 形势判断

| 形×势 | 判定 | 建议 |
|-------|------|------|
| 强+弱 | 蓄势 | 整合资源不急扩张 |
| 强+强 | 发力 | 最佳窗口，集中扩张 |
| 弱+强 | 借势 | 快速融资/合作补形 |
| 弱+弱 | 等待 | 暂停重大投入 |
